# Snake AI

A snake AI written in python. Use curses module, Windows users 
should install it first:

<http://www.lfd.uci.edu/~gohlke/pythonlibs/#curses>


## How to run

	git clone https://github.com/Hawstein/snake-ai.git
	cd snake-ai
	python snake.py
